Perguntas Frequentes

  • O que é html?

HTML (Linguagem de Marcação de Hipertexto) é o código que você usa para estruturar uma página web e seu conteúdo.

  • Onde se usa html?

O HTML é usado em: desenvolvimento frontend, ou seja, na parte visual das aplicações web; desenvolvimento de aplicativos mobile, tanto para Android como para iOS; desenvolvimento de jogos.

  • Qual a diferença entre html e css?

Enquanto que HTML é o preço, tamanhos, descrição, imagem do artigo, etc, e CSS o aspecto da sua loja online, as cores, tipo de letra, tamanho da imagem, quadros e muito mais.
